Reference

Options

Each options are assigned from 1) direct option input, 2) data-attribute. i.e. `dateFrom` can be retrieved as <div class="daterange" data-date-from="[date-from-value]"> </div>
name type desc default
calendar string Year-Month (string) calendar which shows now moment().format('YYYY-MM-DD')
dateFrom string Range selected starts from the date undefined
dateTill string Range selected ends till the date undefined
disabledBefore string The dates before it will be disabled to select undefined
disabledAfter string The dates after it will be disabled to select undefined
onSelected string/function on select handler

function(period, daterange) { /* period (object): period indicator object - from (string): 'YYYY-MM-DD' formatted date that period starts, - till (string): 'YYYY-MM-DD' formatted date that period ends, daterange (jQuery object): daterange instance */ }

(p,dr)=>{ window.alert(p.from,p.till); console.log(dr); }
moveMonth boolean Show Month move control true
moveYear boolean Show Year move control true
titleFormat string moment format to show calendar title 'MMM.YYYY'
weekStart int calendar starting weekday 0 (Sunday)
presets array preset selector.

i.e.
[

  • { title: '이번달', desc: '월초 ~ 현재', range: (till)=>moment(till).date(1).format(DATE_FORMAT) }
  • { title: '일 개월', desc: '30일', range: (till)=>moment(till).add(-30, 'day').format(DATE_FORMAT) }
  • { title: '분기', desc: '3개월 (1/4년)', range: (till)=>moment(till).add(-3, 'month').format(DATE_FORMAT) }
  • { title: '올해', desc: '연초 ~ 현재', range: (till)=>moment(till).dayOfYear(1).format(DATE_FORMAT) }
];
